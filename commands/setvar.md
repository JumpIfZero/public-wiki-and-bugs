## Syntax

```java
SETVAR varName newValue 

varName - Your variable name with exact name match
newValue - New exact value for your variable
```

## Examples

```java
SETVAR myVar 5
SETVAR myVar 2000
SETVAR myVar *logs
SETVAR myVar MyMUleRSN
```
