# {{ page.title }}

_{{ page.description }}_

## Syntax

 
```java
SPECIAL
```
- This command takes no arguments.

## Examples


```java
// Activate the special attack ability
SPECIAL
```

## Notes


- If the player's equipped weapon has no special attack ability, this command will do nothing.
