## Syntax

```java
UNEQUIP    item 

item - ItemName1/ItemID1,ItemName2/ItemID2, ..., ItemNameN/ItemIDN

```

## Examples

```java
UNEQUIP 20997,11864
UNEQUIP armadyl_godsword
UNEQUIP *ancestral*
UNEQUIP *wand*,*staff*
```

## Notes

- Wildcards are supported with this command
- To find itemIDs, you can use kittykeys debug tab or dev tools
