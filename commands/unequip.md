## Syntax

```java
UNEQUIP item 
```

## Examples

```java
UNEQUIP 20997,11864
UNEQUIP armadyl_godsword
UNEQUIP *ancestral*
UNEQUIP *wand*,*staff*
```

## Notes

- To find itemIDs, you can use kittykeys event logger or dev tools
