# {{ page.title }}

_{{ page.description }}_

## Syntax

```java
NPC npc [menuaction] 
```

## Examples

```java
NPC goblin
NPC fishing_spot
NPC perdu 11
NPC 7730 9
```

## Notes

- To find npcIDs, you can use kittykeys event logger or dev tools
- To find menuaction, you can use kittykeys click logger
