## Syntax

```java
BREAK_EVALUATE varName expression 

varName - Your variable name with exact name match
expression - Math expression to evaluate
```

## Examples

```java
BREAK_EVALUATE myVar<=mySecondVar
BREAK_EVALUATE myVar==2+(5*4)
BREAK_EVALUATE myVar!=3
```
