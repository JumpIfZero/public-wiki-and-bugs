## Syntax

```java
MOVE    x y [accuracy] 

x - Number of tiles in x-axis, can be negative
y - Number of tiles in y-axis, can be negative
accuracy - Determines how accurately x/y point should be clicked - unit is tiles, higher number means less accurate

```

## Examples

```java
MOVE 5 1
MOVE -5 -5 2
```
