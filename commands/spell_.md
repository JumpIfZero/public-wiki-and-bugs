# {{ page.title }}

_{{ page.description }}_

## Syntax

```java
SPELL_ [target] 

target - ItemName/ItemID/ObjectName/ObjectID or 1 = Enable left click casting
```

## Examples

```java
ICE_BARRAGE
FIRE_SURGE 1
CHARGE_AIR_ORB 2152
```

## Notes

- To use this command with target action (such as cast to opponent), you must have active target selected
