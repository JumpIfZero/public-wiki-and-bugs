## Syntax

```java
ABORT_ALL ```

## Examples

```java
ABORT_ALL
```

## Notes

{: .warning }
Aborting running scripts may have undetermined behavior. Some scripts may run till completion, while others may stop instantly. Using this command should be avoided.
