## Syntax

```java
TARGET [menuaction] 

menuaction - MenuActionID - leaving menuaction out will default to NPC_SECOND_OPTION (10) for npcs and PLAYER_SECOND_OPTION (45) for players
```

## Examples

```java
TARGET
TARGET 11
```

## Notes

- To find menuaction, you can use kittykeys click logger
