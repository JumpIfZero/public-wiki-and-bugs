## Syntax

```java
WIELD item 
```

## Examples

```java
WIELD 20997,11864
WIELD armadyl_godsword
WIELD *ancestral*
WIELD *wand*,*staff*
```

## Notes

- To find itemIDs, you can use kittykeys event logger or dev tools
