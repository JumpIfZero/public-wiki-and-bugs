## Syntax

```java
TOGGLESCRIPT scriptName [clickMode] 

scriptName - Existing script name, case sensitive exact match
clickMode - 0 = OFF, 1 = ON, leaving clickMode out will always click
```

## Examples

```java
TOGGLESCRIPT 1Tflicker
TOGGLESCRIPT MagePhase 1
```
