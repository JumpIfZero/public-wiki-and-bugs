## Syntax

```java
EVALUATE varName expression 

varName - Your variable name with exact name match
expression - Math expression to evaluate
```

## Examples

```java
EVALUATE myVar myVar+2
EVALUATE myVar 2+(5*4)
EVALUATE myVar Math.max(2000,50)
```
