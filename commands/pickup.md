## Syntax

```java
PICKUP    itemID [telegrab] 

itemID - ItemID1,ItemID2, ..., ItemIDN
telegrab - 1 = telegrab instead of walking

```

## Examples

```java
PICKUP 1240
PICKUP 1240,592,694,7098
PICKUP 2426 1
```

## Notes

- To find itemIDs, you can use kittykeys debug tab or dev tools
