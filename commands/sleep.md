## Syntax

```java
SLEEP    minMS [maxMS] 

minMS - Minimum milliseconds to sleep for
maxMS - Maximum milliseconds to sleep for - leaving maxMS out will sleep exactly for minMS

```

## Examples

```java
SLEEP 2000
SLEEP 50 100
```
