# {{ page.title }}

_{{ page.description }}_

## Syntax

```java
TOGGLEGROUP groupName newStatus 

groupName - Existing group name, case sensitive exact match
newStatus - 0 = OFF, 1 = ON
```

## Examples

```java
TOGGLEGROUP zulrahScripts 1
TOGGLEGROUP MagePhase 0
```
