## Syntax

```java
VARBIT varbitID value 

varbitID - varbit number
value - Minimum value to consider valid for this trigger
```

## Examples

```java
VARBIT 3956 200
```

## Notes

- To find varbitIDs, you can use kittykeys debug tab or dev tools var inspector
