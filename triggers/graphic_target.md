## Syntax

```java
GRAPHIC_TARGET graphicID 
```

## Examples

```java
GRAPHIC_TARGET 85
GRAPHIC_TARGET 4025,694
```

## Notes

- To find graphicIDs, you can use kittykeys event logger or dev tools
- To use this trigger, you must have active target selected
