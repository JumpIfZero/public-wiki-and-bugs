## Syntax

```java
VARPLAYER varplayerID value 

varplayerID - varplayer number
value - Minimum value to consider valid for this trigger
```

## Examples

```java
VARPLAYER 102 1
```

## Notes

- To find varplayerIDs, you can use kittykeys debug tab or dev tools var inspector
