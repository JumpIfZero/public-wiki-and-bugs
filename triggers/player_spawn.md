## Syntax

```java
PLAYER_SPAWN player [minLevel] [maxLevel] 

player - Player name
minLevel - Minimum combat level
maxLevel - Maximum combat level
```

## Examples

```java
PLAYER_SPAWN *
PLAYER_SPAWN myMule123
PLAYER_SPAWN * 116 126
```
