## Syntax

```java
STAT skillName maxLevel [minLevel] 
```

## Examples

```java
STAT ATTACK 95
STAT HITPOINTS 90 60
STAT HITPOINTS 60 40
```

## Notes

- When using skill name, double check that the name is right. For example runecraft instead of runecrafting
