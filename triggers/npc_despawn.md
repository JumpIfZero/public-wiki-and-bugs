_{{ page.description }}_

## Syntax

```java
NPC_DESPAWN npc 

npc - NpcName1/NpcID1,NpcName2/NpcID2, ..., NpcNameN/NpcIDN
```

## Examples

```java
NPC_DESPAWN cerberus
NPC_DESPAWN 2790,2792
```

## Notes

- To find npcIDs, you can use dev tools
