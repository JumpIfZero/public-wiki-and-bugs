## Syntax

```java
ITEM_SPAWN    item 

item - ItemName1/ItemID1,ItemName2/ItemID2, ..., ItemNameN/ItemIDN

```

## Examples

```java
ITEM_SPAWN 12819
ITEM_SPAWN 12819,12849,59286
```

## Notes

- To find itemIDs, you can use kittykeys debug tab or dev tools
