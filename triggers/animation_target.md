## Syntax

```java
ANIMATION_TARGET animationID 
```

## Examples

```java
ANIMATION_TARGET 104
ANIMATION_TARGET 705,599
```

## Notes

- To find animationIDs, you can use kittykeys event logger or dev tools
- To use this trigger, you must have active target selected
