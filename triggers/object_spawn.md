## Syntax

```java
OBJECT_SPAWN object 
```

## Examples

```java
OBJECT_SPAWN 9036,9033
OBJECT_SPAWN bank_booth
```

## Notes

- To find objectIDs, you can use kittykeys event logger or dev tools
- Objects include game objects, wall objects, ground objects and decorative objects
