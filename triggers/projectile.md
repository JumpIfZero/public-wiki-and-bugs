## Syntax

```java
PROJECTILE projectileID 

projectileID - ProjectileID1,ProjectileID2, ..., ProjectileIDN
```

## Examples

```java
PROJECTILE 2948
PROJECTILE 2948,2949,2950
```

## Notes

- To find projectileIDs, you can use kittykeys debug tab or dev tools
