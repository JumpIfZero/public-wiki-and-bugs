## Syntax

```java
GRAPHIC_SELF    graphicID 

graphicID - graphicID1,graphicID2, ..., graphicIDN

```

## Examples

```java
GRAPHIC_SELF 345
GRAPHIC_SELF 345,597
```

## Notes

- To find graphicIDs, you can use kittykeys debug tab or dev tools
