## Syntax

```java
SOUND_EFFECT soundID 
```

## Examples

```java
SOUND_EFFECT 2885
SOUND_EFFECT 596,39,63
```

## Notes

{: .warning }
You will need to have effect sounds at minimum of 1% volume to get sound triggers
